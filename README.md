# ALFS Detection

## Setup

This repository is using code from the [alfs_py](https://github.com/bambi-eco/alfs_py) project.

So activate your environment and install alfs_py locally since it is not available on pypi:

```
pip install <path-to-alfs_py-project>
```

In addition, also parts of the [Pipeline](https://github.com/bambi-eco/Pipeline) project are used.

So again activate your environment and install Pipeline locally since it is not available on pypi:

```
pip install <path-to-pipeline-project>
```

## Usage

Use the provided script and provide the required input data.
Afterwards just hit run ;)
